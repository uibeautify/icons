# Contributing

UI Beautify Icons is currently maintained as a closed, owner-managed icon library.

At this time, public contributions, pull requests, icon submissions, and style changes are not accepted.

If you find an issue, need a feature, have a suggestion, Want to parise :) or want to request an icon, 

You can do this on this [Submit Feedback](https://icons.uibeautify.com/feedback) page
