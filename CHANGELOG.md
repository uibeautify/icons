# Changelog

## [0.1.1] - 2026-06-02 `Current`

### Added

* Initial release of UI Beautify Icons.
* Added the free React package: `@uibeautify/icons-react-free`.
* Added the Pro React package as a downloadable `.tgz` file.
* Added Outline icon support.
* Added Solid icon support for Pro users.
* Added Duotone icon support for Pro users.
* Added Duocolor icon support for Pro users.
* Added customizable React props:

  * `size`
  * `variant`
  * `color`
  * `strokeWidth`
  * `secondaryOpacity`
  * `secondaryColor`
* Added default icon props:

  * `variant="outline"`
  * `size={24}`
  * `color="#000000"`
  * `secondaryOpacity={0.15}`
  * `secondaryColor="#999999"`
* Added support for importing icons as React components.
